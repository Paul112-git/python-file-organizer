# python-file-organizer
A python script that automatically organizes files into folders by typeT

## Features

-Moves images to photos
-Moves videos to Videos
-Moves music to music
-Moves documents to Documents
-Move archives to Archives
-Show the number of the files moved
-Display the total files organized

## Technologies

-Python
-os
-shutil

## How to Run

1. run the script
2. Enter the folder path
3. The files are automatically organized.
